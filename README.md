<h3>How to use?</h3>

You'll have to clone the Fillit repo by taping :

        git clone https://github.com/temun17/Fillit.git

Then you'll have to compile the project :

        > cd fillit
        
        > make
        
Now you can run <code>./fillit <tetriminos_file></code>
